# Changelog

## 0.1.0 (2026-03-22)

### Features

- Initial release
- Purpose-driven architecture linting protocol
- CLI commands: `init`, `validate`, `run`, `list`
- 6 built-in presets: billing, layer-boundary, race-condition, auth-boundary, ai-code-quality, test-integrity
- Cascading config lookup (monorepo support)
- Interactive purpose selection UI
- Multiple output formats: prompt, json, markdown
- Agent-agnostic prompt generation
